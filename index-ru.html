Добро пожаловать на сайт проекта DMO Advanced Launcher

Привет! Хотим представить вашему вниманию первый пользовательский Лаунчер для Digimon Masters Online! Мы вложили все усилия и душу в его создание, он совсем не похож на стандартный Лаунчер. Он: красивее, удобнее, функциональнее.

Обзор и возможности

DMO Advanced Launcher поддерживает ADMO (Aeria), GDMO (GameKing), KDMO (www.digimonmasters.com and IMBC) серверы. Однако, не все возможности Лаунчера поддерживаются всеми серверами. Например, ДигиРотация и страница сообщества недоступны в режиме работы для сервера Aeria. Функциональность распределена по разным категориям:

A
- поддерживается в режиме Aeria
G
- поддерживается в режиме GameKing
K
- поддерживается в режиме Korean (оба www.digimonmasters.com и IMBC)
S
- поддерживается в любом режиме

Системные требования

  • Windows Vista SP2 / Windows 7 SP1 / Windows 8 / Windows 8.1 / Windows 10
  • Microsoft .NET Framework 4.5

Помощь в переводе

Мы рады получить любую помощь в переводе нашего Лаунчера на другие языки. Обратите внимание на папку Languages в корне Лаунчера. Используйте файл Template_for_translation.txt как шаблон. Пожалуйста, отправляйте ваши переводы на нашу почту.

Известные проблемы

  • Страница сообщества и ДигиРотация используют информацию с веб-сайта игры. Не все Дигимоны предоставлены в рейтинге или страницах Теймеров (особенно у Failmax Joymax GameKing). Поэтому их не будет и в Лаунчере.
  • К тому же, только ОДИН Дигимон ОДНОГО типа может быть получен с сайта игры. Так что, если у Вас, к примеру, два Импмона, то в Лаунчере будет только один из них.
  • Долгий процесс авторизации для KDMO IMBC. Ошибка 705 - это нормально, просто ждите успешного входа. Если вы получаете эту ошибку более десяти раз, проверьте данные вашей учетной записи.
