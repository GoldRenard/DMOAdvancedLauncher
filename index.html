Welcome to DMO Advanced Launcher GitHub Page.

Hello! We would like to present to you the custom Digimon Masters Online* launcher. We invested all our effort and soul into its creation. It doesn't look like a default launcher. It is: beautiful, comfortable, functional.

Overview and features

DMO Advanced Launcher supports ADMO (Aeria), GDMO (Joymax), KDMO (www.digimonmasters.com and IMBC) servers. Unfortunately, not all features are supported for all servers. For example, Digimon Rotation and Community page not available for ADMO.

System requirements

  • Windows® XP (SP3) / Vista (SP1+) / 7 / 8
  • Microsoft .NET Framework 4.0
  • Microsoft Visual C++ 2010 SP1 Redistributable Package

Installation

  1. Download and install the latest release.
  2. If you have Windows XP and want use AppLocale, you MUST install East Asian Language support (instruction), then turn on the AppLocale in options of launcher.

Known issues

  • Community page and DigiRotation uses information of game website. You know, not all digimons are provided at ranking or tamer's information pages (especially at Joymax). So not all digimons provided at our launcher
  • In additional, launcher can read only ONE digimon of ONE type from website. So if you have two Impmons then only ONE will be added to database of launcher. Usually it's a latest hatched Digimon
  • Long logon process for KDMO IMBC. Error 705 is normal, you must just wait for successful login. If you will get error 705 10 or more times, check your account data

How to get Steam Overlay work with AdvancedLauncher

There are two things that prevent game launching through Steam + AdvancedLauncher with working In-game status and Overlay.

  • First, Steam must be runned AS ADMIN.
  • Second, Steam can't properly detect the game if it was launched through AppLocale. So Advanced Launcher will disable AppLocale if it launched through Steam. If you want use AppLocale, you must run Steam through AppLocale:
    1. Open Microsoft AppLocale Wizard ("AppLocale" in the Start Menu)
    2. In "Locate The Application" screen click "Browse..." and choose Steam.exe
    3. Choose Korean (한국어) on the next screen
    4. On the next screen check "Create a Shortcut to always run this application with AppLocale" and click Finish
    5. The shortcut should be in the folder of the Start Menu where AppLocale is. Change it to always run Steam as Administrator in Properties > Compatibility tab. You can also make it run this way for all users if you want
    6. Click Ok and run it. You can also copy the shortcut to the Desktop or Pin it to the Start Menu or Task Bar for easy access
    7. In-game status and Overlay should work now
