LockedChangedHandler AdvancedLauncher.SDK.Management.IEnvironmentManager.ClosingLocked

ClosingLocked event, fires on application close locking. You should not use this directly, is doesn't work correctly for cross-domain transparent proxy instances. Use ClosingLockedProxy(EventProxy<LockedEventArgs>, bool) instead.

LockedChangedHandler AdvancedLauncher.SDK.Management.IEnvironmentManager.FileSystemLocked

FileSystemLocked event, fires on game archives locking. You should not use this directly, is doesn't work correctly for cross-domain transparent proxy instances. Use FileSystemLockedProxy(EventProxy<LockedEventArgs>, bool) instead.
